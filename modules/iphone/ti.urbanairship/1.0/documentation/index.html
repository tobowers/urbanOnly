urbanairship Module

Description

The UrbanAirship module is a way to connect to the UrbanAirship Airmail push notification service, and retrieve messages from it.

Accessing the urbanairship Module

To access this module from JavaScript, you would do the following:

var urbanairship = require("ti.urbanAirship");

The urbanairship variable is a reference to the Module object.

Reference

Functions

ti.urbanAirship.registerDevice({...})

Registers the device with UrbanAirship. This operation is only performed once, information is automatically stored by the UrbanAirship module. Note that if a user deletes and reinstalls your app, it will re-register with UrbanAirship.

Arguments

Takes one argument, a dictionary with keys:

token[object]: A blob object containing the device's token. This should be obtained by a Ti.Network.registerForPushNotifications() callback.
alias[string] (optional): A string identifying the device's alias.

ti.urbanAirship.displayInbox([{...}])

Displays the inbox for a user. This can either be the autogenerated 'default' inbox (keyed to this specific device) or an inbox which has been set up through the UrbanAirship API (not yet supported by the UA module).

The inbox is displayed via a picker that allows users to browse and display messages they have recieved.

Arguments

Takes an optional argument, a dictionary with keys:

user[string]: The user to retrieve the inbox for
password[string]: The user's password
alias[string]: The alias to associate with the user
tags[array]: The tags for the inbox

ti.urbanAirship.hideInbox()

Hide the inbox picker if it is currently displayed.

Properties

ti.urbanAirship.appKey

The app key generated by UrbanAirship for your app.

ti.urbanAirship.secret

The app secret generated by UrbanAirship for your app. Note that this is not your master secret.
